HELP FOR NOVASYON QUIZ


1 - WHAT IS NOVASYON QUIZ?
Novasyon Quiz is a game based upon the poplar jeopardy game. We want the settings to be easy, so you can fully enjoy it.
We will have a lot of quizzes. But we started with Bible quizzes. Later on, we will have other types of quizzes: history, sciences, languages.
2 - HOW DO WE CALCULATE THE SCORE?
The scoring is based on the setting. Setting your preferred settings according to difficulty.
3 - PLAYING BIBLE QUIZ
i - ENTERING NAMES AND SELECTING OPTIONS Enter the player’s name and level on this screen. There are 5 suggested levels:
a) Very easy – As the name suggest.
b) Easy
c) Medium (The default level)
d) Hard
e) Surprise! Anything goes here. Select questions from all of the above categories.

ii - PLAYER'S NAME, LEVEL
Enter the name of each player and choose a level. The default level is Medium
iii - GAME SETTINGS
These modes allow you to find the most interesting and exciting method of playing the quiz
I) 2 tries; no penalty for wrong answer. Hints take one turn.
2 opportunities to answer the question with no penalty for wrong answers. However, a hint uses 1 of your tries.
In this mode, the player can try 2 times to answer a question. If the first try is wrong, the player will be able to make a second try. If the second try is still wrong, the player will not lose any points.
NB: The player will be able to see the hint only before the 1st try. If the player uses the hint (if hint is available for this answer) the player will not be able to make a second, try.
